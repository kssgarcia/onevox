01_INSTALLATION_GUIDE

macOS

Universal binary (ARM64/x86_64) 

curl -fsSL https://raw.githubusercontent.com/kssgarcia/onevox/main/install.sh | sh

REQUIRED PERMISSIONS:

  • Input Monitoring (Hotkey)
  • Accessibility (Injection)
  • Microphone (Audio)

Linux

Systemd / Binary install 

curl -fsSL https://raw.githubusercontent.com/kssgarcia/onevox/main/scripts/install_linux.sh | bash

SETUP:

sudo usermod -aG audio,input $USER && systemctl --user enable --now onevox

Troubleshooting

Hotkey not working? On macOS, you must restart the daemon after granting permissions: launchctl kickstart -k gui/$(id -u)/com.onevox.daemon

Wayland Issues? Wayland security prevents global hotkey detection. Bind onevox start-dictation and stop-dictation manually to your compositor keys.

No Audio? Run onevox test-audio --duration 3 to verify hardware access.
